对应参数：`scad/parameters.scad`。  
坐标系约定见下文。

| 项目 | 数值 | 备注 |
|------|------|------|
| 长度 | **32 mm** | 插针拆除后仍按 32 mm 板长 |
| 宽度 | **12 mm** | |
| 厚度 | **1.6 mm** | 典型 FR4，未在图上标注 |

## 2. 自左向右布局（原点 = 板左缘，单位 mm）

板宽方向二者均**水平居中**（距上下边各 6 mm，即板宽中线）。

```
板左缘 0                                                    32 板右缘
|-------|---------|-----------|------------|------|--------|
|       | 10~15   |           | 18.5~27    |      | 焊盘区 |
|       | 电位器轴 |           | 数码管      |      | (去插针)|
|-------|---------|-----------|------------|------|--------|
              ↕ 板宽居中              ↕ 板宽居中
```

| 区间（自板左缘） | 元件 | 跨度 | 中心（自板左缘） |
|------------------|------|------|------------------|
| **10 ~ 15** | 电位器轴 | 5 mm | **12.5** |
| **18.5 ~ 27** | 数码管 | 8.5 mm | **22.75** |

### 换算到 OpenSCAD 板心坐标

原点 = 板心，X 向右，Y 向上（板宽中线 Y=0）：

| 元件 | 板心 X | 板心 Y | 开窗/外形 |
|------|--------|--------|-----------|
| 电位器轴 | **-3.5** | **0** | 壳孔 ⌀ **6.5**（轴≈⌀5，加大消化打印公差）；旋钮帽壳外再装 |
| 数码管 | **6.75** | **0** | 外形 **8.5 × 12**（短边沿板长，**长边 12** 沿板宽） |

- 轴：`X = (10+15)/2 - 16 = -3.5`
- 数码管：`X = (18.5+27)/2 - 16 = 6.75`，`SIZE_X = 27-18.5 = 8.5`，`SIZE_Y = 12`

### 高度（Z）

| 项目 | 数值 | 说明 |
|------|------|------|
| 旋钮腔内净高 `KNOB_CAVITY_H` | **7.5 mm** | PCB 顶面（分型面）→ 顶盖内壁；保证装板不晃、轴能露出 |
| 数码管壳内高度 `DISPLAY_HEIGHT_ABOVE` | **6.5 mm**（可调） | 勿超过腔内净高 7.5 |
| 旋钮帽 | 壳外再装 | 壳上只留轴孔，不按帽外径开大孔 |
| 轻触开关 | — | **不开孔** |

## 3. 安装孔（对角，距顶点约 1.5 mm）

孔心相对所在角的 **X、Y 边各内缩约 1.5 mm**（`MODULE_HOLE_INSET = 1.5`）。

照片坐标：原点 = 板**左上角**，X 右，Y 下。

| 孔位 | 相对顶点 | 照片 (X, Y) | 板心坐标 |
|------|----------|-------------|----------|
| 左上 | 距左、距上各 **1.5** | **(1.5, 1.5)** | **(-14.5, +4.5)** |
| 右下 | 距右、距下各 **1.5** | **(30.5, 10.5)** | **(+14.5, -4.5)** |

| 项目 | 数值 |
|------|------|
| 孔径 | ≈ **2.0 mm**（M2） |
| 孔心距 | ΔX = 29 mm，ΔY = 9 mm |

用途：盒内螺丝穿过 PCB 锁入下半**罗马柱**（不从壳外拧这两颗）。

## 4. 壳体（与模组配合）

| 项目 | 数值 | 说明 |
|------|------|------|
| 内腔长 × 宽 × 高 | **70 × 22 × 15 mm** | 上半净高 7.5；加长以免罗马柱与线缆立柱冲突 |
| 壁厚 | **1.6 mm** | |
| 外廓（约） | **73.2 × 25.2 × 18.2 mm** | 内腔 + 2×壁厚 |
| 合盖螺丝 | 壳外 **4× M2** | 四角 Boss（`BOSS_X=28`, `BOSS_Y=7.5`） |
| 模组固定 | 盒内 **2× 罗马柱** | 对齐上表安装孔 |
| 线缆 | ⌀5 mm，两端出线 + 锯齿/立柱夹持 | |
