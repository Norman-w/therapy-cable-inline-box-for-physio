## 2. 自左向右布局（原点 = 板左缘，单位 mm）

板宽方向二者均**水平居中**（距上下边各 6 mm，即板宽中线）。

```
板左缘 0                                                 31.5 板右缘
|-------|---------|-----------|------------|------|--------|
|       | 10~15   |           | 18.5~27    |      | 焊盘区 |
|       | 电位器轴 |           | 数码管      |      | (去插针)|
|-------|---------|-----------|------------|------|--------|
              ↕ 板宽居中              ↕ 板宽居中
```

| 区间（自板左缘） | 元件 | 跨度 | 中心（自板左缘） |
|------------------|------|------|------------------|
| **10 ~ 15** | 电位器轴 | 5 mm | **12.5** |
| **18.5 ~ 27** | 数码管 | 8.5 mm | **22.75** |

### 换算到 OpenSCAD 板心坐标

原点 = 板心（半长 15.75），X 向右，Y 向上（板宽中线 Y=0）：

| 元件 | 板心 X | 板心 Y | 开窗/外形 |
|------|--------|--------|-----------|
| 电位器轴 | **-3.25** | **0** | 壳孔 ⌀ **8.0**；旋钮帽壳外再装 |
| 数码管 | **7.0** | **0** | 外形 **8.5 × 12**（长边 12 沿板宽） |

- 轴：`X = 12.5 - 15.75 = -3.25`
- 数码管：`X = 22.75 - 15.75 = 7.0`

### 高度（Z）

| 项目 | 数值 | 说明 |
|------|------|------|
| 旋钮腔内净高 `KNOB_CAVITY_H` | **7.5 mm** | PCB 顶面（分型面）→ 顶盖内壁；保证装板不晃、轴能露出 |
| 数码管壳内高度 `DISPLAY_HEIGHT_ABOVE` | **6.5 mm**（可调） | 勿超过腔内净高 7.5 |
| 旋钮帽 | 壳外再装 | 壳上只留轴孔，不按帽外径开大孔 |
| 轻触开关 | — | **不开孔** |

## 3. 安装孔（实物复测）

| 方向 | 孔心到边 | 说明 |
|------|----------|------|
| 短边（左右端） | **1.5 mm** | `MODULE_HOLE_INSET_SHORT` |
| 长边（上下边） | **2.0 mm** | `MODULE_HOLE_INSET_LONG` |

照片坐标：原点 = 板**左上角**，X 右，Y 下；板长 **31.5**。

| 孔位 | 相对边距 | 照片 (X, Y) | 板心坐标 |
|------|----------|-------------|----------|
| 左上 | 左 1.5 / 上 2.0 | **(1.5, 2.0)** | **(-14.25, +4.0)** |
| 右下 | 右 1.5 / 下 2.0 | **(30.0, 10.0)** | **(+14.25, -4.0)** |

| 项目 | 数值 |
|------|------|
| 孔径 | ≈ **2.0 mm**（M2） |
| 孔心距 ΔX | **28.5 mm**（= 31.5 − 1.5 − 1.5，与实测一致） |
| 孔心距 ΔY | **8.0 mm**（= 12 − 2 − 2） |
| 对角距离 | ≈ **29.6 mm** |

> 旧版按 32×双内缩 1.5 → ΔX=29、对角≈**30.4**，与打印≈30 / 板孔≈28.5 对不上；已按上表修正。

用途：盒内螺丝穿过 PCB 锁入下半**罗马柱**。
