# Synairy ESC-Adaptor V02（扩展输入版）

v2 方案所用直流电机 / 电调 PWM 调速信号板。资料来源见 [`images/`](images/)。

插针（`SIG` / `VIN` / `GND`）装配前拆除，改焊线引出。

**完整尺寸表（PCB / 开孔 / 壳体）→ [`dimensions.md`](dimensions.md)**

## 尺寸速查

| 项目 | 数值 |
|------|------|
| PCB | **32 × 12 × 1.6 mm** |
| 自左向右 | 轴 **10~15**；数码管 **18.5~27**；均板宽居中 |
| 数码管外形 | **8.5 × 12**（长边 12） |
| 轴孔 / 腔内柱高 | 壳 ⌀**6.5**；腔内净高 **7.5 mm**（帽壳外再装） |
| 开关 | **不开孔** |
| 安装孔（板心） | 距对角顶点各 **1.5**：(-14.5, 4.5) / (14.5, -4.5)，M2 |
| 壳体内腔 | **70 × 22 × 15 mm**；外 4×M2 合盖 + 内 2×罗马柱锁板 |

## 接口

| 丝印 | 含义 |
|------|------|
| `SIG` | PWM 信号输出 |
| `VIN` | 供电正极 |
| `GND` | 地 |

板底另有焊盘：`VDD` / `IN` / `GND`（扩展输入）。

## 电气与 PWM

| 项目 | 规格 |
|------|------|
| 工作电压 | DC **3–24 V** |
| 接口 | 3P：供电 + PWM 输出 |
| 显示 | 数码管，转速百分比 |
| 伺服 / 无刷电调模式 PWM 频率 | **50 Hz**（固定） |
| 脉宽调节范围 | **0.95 ms – 1.95 ms** |

## 操作与模式

板载：黄色旋钮调速；轻触键用于开关启动（见 [`images/03.webp`](images/03.webp)）。

焊盘短接模式见 [`images/02.webp`](images/02.webp)：

| 模式 | 配置 | 行为 |
|------|------|------|
| 默认模式 | 焊盘空置 | 旋钮直接调速 |
| 上电启动 | 短接「上电启动」 | 上电约 3 s 后爬升到设定转速 |
| 开关启动 | 短接「开关启动」 | 需按住按键/闭合开关才运转 |
