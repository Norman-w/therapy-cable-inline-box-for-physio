# Synairy ESC-Adaptor V02（扩展输入版）

v2 方案所用直流电机 / 电调 PWM 调速信号板。资料来源见 [`images/`](images/)。

## 外形尺寸

| 项目 | 数值 | 来源 |
|------|------|------|
| PCB 长度 | **32 mm** | [`images/04.webp`](images/04.webp) 标注 |
| PCB 宽度 | **12 mm** | [`images/04.webp`](images/04.webp) 标注 |
| 排针 | 右侧 3P 直角排针，超出 32 mm 板长 | 照片 |
| 排针间距 | 2.54 mm（标准） | 布局推断 |
| 安装孔 | 对角两孔（左上 / 右下） | 照片 |

> 壳体设计时需额外预留：旋钮高度、数码管高度、直角排针伸出长度。

## 接口

| 丝印 | 含义 |
|------|------|
| `SIG` | PWM 信号输出 |
| `VIN` | 供电正极 |
| `GND` | 地 |

板底另有焊盘：`VDD` / `IN` / `GND`（扩展输入）。

## 电气与 PWM

| 项目 | 规格 |
|------|------|
| 工作电压 | DC **3–24 V** |
| 接口 | 3P：供电 + PWM 输出 |
| 显示 | 3 位数码管，转速百分比 |
| 伺服 / 无刷电调模式 PWM 频率 | **50 Hz**（固定） |
| 脉宽调节范围 | **0.95 ms – 1.95 ms** |

## 操作与模式

板载：

- 黄色旋钮电位器：调节转速百分比
- 轻触按键：开关启动（见 [`images/03.webp`](images/03.webp)）

通过板上焊盘短接切换模式（见 [`images/02.webp`](images/02.webp)）：

| 模式 | 配置 | 行为 |
|------|------|------|
| 默认模式 | 所有焊盘空置 | 旋钮直接调速；按键/外部开关无效 |
| 上电启动 | 短接「上电启动」 | 上电约 3 s 后自动爬升到旋钮设定转速 |
| 开关启动 | 短接「开关启动」 | 上电不转；需按住按键或闭合外部开关才保持运转，松开则减速停止 |
