# Synairy ESC-Adaptor V02（扩展输入版）

v2 方案所用直流电机 / 电调 PWM 调速信号板。资料来源见 [`images/`](images/)。

插针（`SIG` / `VIN` / `GND`）装配前拆除，改焊线引出。

## 外形尺寸

| 项目 | 数值 | 来源 |
|------|------|------|
| PCB 长度 | **32 mm** | [`images/04.webp`](images/04.webp) 标注 |
| PCB 宽度 | **12 mm** | [`images/04.webp`](images/04.webp) 标注 |

## 安装孔（图像素标定）

以 **PCB 左上角为原点、X 向右、Y 向下**（与 [`images/04.webp`](images/04.webp) 一致）：

| 孔位 | 中心 (X, Y) mm | 相对边缘 |
|------|----------------|----------|
| 左上 | **(3.5, 2.6)** | 距左 ≈3.5，距上 ≈2.6 |
| 右下 | **(29.2, 10.1)** | 距右 ≈2.8，距下 ≈1.9 |

换算到 **板心坐标**（OpenSCAD 使用，X 右、Y 上）：

| 孔位 | 板心坐标 mm |
|------|-------------|
| 左上 | **(-12.50, +3.40)** |
| 右下 | **(+13.20, -4.10)** |

| 项目 | 数值 |
|------|------|
| 孔间距 | ΔX ≈ 25.7 mm，ΔY ≈ 7.5 mm |
| 孔径 | ≈ **2.0 mm**（M2） |
| 焊盘外径 | ≈ 2.4–2.6 mm |

## 旋钮 / 数码管 / 按键（开窗用）

板心坐标：

| 元件 | 中心 mm | 外形 | 开窗建议 |
|------|---------|------|----------|
| 电位器轴 | 板左缘 **10~15**（心 **-3.5, 0**） | 板宽居中；腔内柱高 **7.5** | 壳开 ⌀**5.2**，帽壳外再装 |
| 数码管 | 板左缘 **18.5~27**（心 **6.75, 0**） | **8.5×12**（长边 12） | 高度 `DISPLAY_HEIGHT_ABOVE` 可调 |
| 轻触键 | 板载有 | — | **不开孔** |

> 装配：盒内 M2 穿过 PCB 锁入罗马柱；盒外另用四角 4× M2 合盖。

## 接口

| 丝印 | 含义 |
|------|------|
| `SIG` | PWM 信号输出 |
| `VIN` | 供电正极 |
| `GND` | 地 |

板底另有焊盘：`VDD` / `IN` / `GND`（扩展输入）。

## 电气与 PWM

| 项目 | 规格 |
|------|------|
| 工作电压 | DC **3–24 V** |
| 接口 | 3P：供电 + PWM 输出 |
| 显示 | 数码管，转速百分比 |
| 伺服 / 无刷电调模式 PWM 频率 | **50 Hz**（固定） |
| 脉宽调节范围 | **0.95 ms – 1.95 ms** |

## 操作与模式

板载：黄色旋钮调速；轻触键用于开关启动（见 [`images/03.webp`](images/03.webp)）。

焊盘短接模式见 [`images/02.webp`](images/02.webp)：

| 模式 | 配置 | 行为 |
|------|------|------|
| 默认模式 | 焊盘空置 | 旋钮直接调速 |
| 上电启动 | 短接「上电启动」 | 上电约 3 s 后爬升到设定转速 |
| 开关启动 | 短接「开关启动」 | 需按住按键/闭合开关才运转 |
